# YazCar

تطبيق YazCar لإدارة معلومات السيارات والصيانة

## الاستضافة

تم استضافة النسخة الويب من هذا التطبيق على [Vercel](https://vercel.com)

## طريقة تشغيل المشروع محلياً

### تثبيت التبعيات
```bash
npm install
# أو
yarn install
```

### تشغيل التطبيق
```bash
# لتشغيل نسخة الويب
npm run web
# لتشغيل نسخة الأندرويد
npm run android
# لتشغيل نسخة الآيفون
npm run ios
```

### بناء نسخة الويب
```bash
npm run build
```
